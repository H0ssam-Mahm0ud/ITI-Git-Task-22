##Annotated tags VS Lightweight tags

to create a lightweight tag

git tag v1.0


to create an annotated tag

git tag -a v1.7


to list all tags 

git tag

deleting a local tag 

git tag -d v1.7

deleting a remote tag

git push origin --delete v1.7
